MAC Drop Commands

Port Security / MAC Violation / Errdisable / MAC Learning 

show port-security
show port-security interface fa0/1
show mac address-table
show mac address-table interface fa0/1
show interfaces status err-disabled
show errdisable recovery
show logging

show port-security → ดู Port Security ทั้งหมดใน Switch

show port-security interface fa0/1 → ดู MAC Violation ของ Port

show mac address-table → ดู MAC Address ที่เรียนรู้

show mac address-table interface fa0/1 → ดู MAC บน Port ที่ต้องการ

show interfaces status err-disabled → ดู Port ที่ถูก Disable

show errdisable recovery → ดูระบบ Auto Recovery ของ Port

show logging → ดู Log Security / Violation

ใช้กับ: Cisco Switch
