IP Block Commands

ACL / Firewall / NAT / IP Restriction 

show access-lists
show ip access-lists
show run | include access-group
show run | section access-list
show ip nat translations
show ip nat statistics
show conn
show logging

show access-lists → ดู ACL ที่ใช้ Block / Allow IP

show ip access-lists → ดู ACL แบบละเอียด

show run | include access-group → ดูว่า ACL Apply ที่ Interface ไหน

show run | section access-list → ดู Config ACL เต็ม

show ip nat translations → ดู NAT Session

show ip nat statistics → ดู NAT Statistics

show conn → ดู Firewall Connection

show logging → ดู Log การ Block IP

ใช้กับ: Router / Firewall / L3 Switch
