Security Commands

ACL / Port Security / Login / MAC Block / IP Block / Firewall

ACL & IP Block 

show access-lists
show ip access-lists
show run | include access-group
show run | section access-list

show access-lists → ดู ACL ทั้งหมดที่ใช้ Block / Allow IP

show ip access-lists → ดู ACL แบบละเอียด พร้อม Hit Count

show run | include access-group → ดูว่า ACL ถูก Apply ที่ Interface ไหน

show run | section access-list → ดู Config ACL เต็มทั้งหมด

ใช้กับ: Router / L3 Switch / Firewall

Port Security / MAC Drop 

show port-security
show port-security interface fa0/1
show mac address-table
show interfaces status err-disabled
show errdisable recovery

show port-security → ดูสถานะ Port Security ทุก Port

show port-security interface fa0/1 → ดู MAC ที่ถูก Block หรือ Violation

show mac address-table → ดู MAC Learning บน Switch

show interfaces status err-disabled → ดู Port ที่โดน Shutdown จาก Security

show errdisable recovery → ดูระบบ Auto Recovery ของ Port

ใช้กับ: Cisco Switch

User & Login Security 

show users
show login
show ssh
show aaa sessions

show users → ดู User ที่ Login อยู่

show login → ดู Login Status

show ssh → ดู SSH Session

show aaa sessions → ดู AAA Authentication Session

ใช้กับ: Router / Switch

Firewall / NAT 

show conn
show xlate
show ip nat translations
show ip nat statistics

show conn → ดู Connection ที่ผ่าน Firewall

show xlate → ดู NAT Translation บน ASA

show ip nat translations → ดู NAT Session

show ip nat statistics → ดู NAT Statistics

ใช้กับ: Cisco ASA / Router

Security Troubleshooting 

ping
traceroute
show logging
debug ip packet

ping → ทดสอบ Connectivity

traceroute → ดูเส้นทาง Packet

show logging → ดู Security Logs

debug ip packet → Debug Packet แบบ Real-time

ใช้กับ: Router / Firewall
