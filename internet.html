ping 8.8.8.8
→ ทดสอบว่าออก Internet ได้หรือไม่

ping gateway-ip
→ ทดสอบ Gateway ว่ายังติดต่อได้

traceroute 8.8.8.8
→ ตรวจสอบเส้นทาง Network

show ip interface brief
→ ดู Interface Up/Down

show access-lists
→ ตรวจสอบ ACL ว่ามีการ Block IP หรือไม่

show mac address-table
→ ตรวจสอบว่า MAC ของเครื่องเรียนรู้บน Switch หรือไม่

show arp
→ ตรวจสอบ ARP Table

show interfaces status
→ ดูสถานะ Port

show interfaces counters errors
→ ตรวจสอบ CRC / Error / Drops

show port-security
→ ตรวจสอบว่า MAC ถูก Security Block หรือไม่

show logging
→ ดู Logs Error ล่าสุด

show spanning-tree
→ ตรวจสอบว่า Port ถูก STP Block หรือไม่

show vlan brief
→ ตรวจสอบ VLAN ของ Port

show ip route
→ ตรวจสอบ Default Route

show ip dhcp binding
→ ตรวจสอบ DHCP Lease

show ip dhcp conflict
→ ตรวจสอบ IP Conflict

ใช้งานกับ: Cisco Router / Switch / Firewall
